Inroduction

Web development involves creating and maintaining websites. It covers various aspects, including web design, web programming, and database management.

HTML

HTML (HyperText Markup Language) is the standard language used to create and design web pages. It structures content with elements like headings, paragraphs, and links.

Example:

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the look and feel of a website, including colors, fonts, and spacing.

Example:

body {
    background-color: #f0f0f0;
    font-family: Arial, sans-serif;
    }
h1 {
    color: #333;
    }

JavaScript

JavaScript is a programming language that enables interactive web pages. It can manipulate the DOM, handle events, and make web pages dynamic.

Example:

document.getElementById('myButton').addEventListener('click', function() {
    alert('Button clicked!');
});
